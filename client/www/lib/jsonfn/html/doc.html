Documentation

  • Installation
    • Browser ( all major browsers and IE9+ ) 
      <script type="text/javascript" src="path/to/jsonfn.js"></script>
    • Node.js 
      npm install json-fn
  • Initialization
    • Browser: 
      not needed
    • Node.js: 
      var JSONfn = require('path/to/json-fn');
  • Usage:
    • Stringify 
      var str = JSONfn.stringify(obj);
      @obj - javascript Object;
      @return- String;
    • Parse 
      var obj = JSONfn.parse(str[, date2obj]);
      @str - string, produced with JSONfn.stringify() function;
      @date2obj- Boolean (optional); Set it true if original object had property instance of Date.
      @return- javascript Object;
    • Clone 
      var objClone = JSONfn.parse(obj[, date2obj]);
      @obj - javascript Object;
      @date2obj- Boolean (optional); Set it true if original object had property instance of Date.
      @return- javascript Object;
  • Testing
    • Browser ( add this to header and see result in console ) 
      <script type="text/javascript" src="test/jsonfn_test.js"></script>
    • Node.js
      >node test/jsonfn_test.js
