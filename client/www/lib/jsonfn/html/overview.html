Overview

JSONfn is a javascript plugin to stringify | parse | clone a javascript object.

With this plugin you can serialize javascript object with such property as Function, RegExp, Date, and re-create original object from this string.

JSONfn Advantage:
Property Type JSON JSONfn Comments
Basic YES YES
Function - YES Regular JSON ignores functions.
RegExp - YES Regular JSON returns empty object.
Date - YES Regular JSON returns a "String" instead of "Date" object.

Plugin is developed to work in both Browser and Node.js environment.

Plugin is very compact: full development version (with comments, copyright, license, etc.) is about 2k
